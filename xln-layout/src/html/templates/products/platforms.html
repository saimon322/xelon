Computing

Unsere in der Schweiz entwickelte und gehostete Cloud-Plattform unterstützt IT-Dienstleister beim Ausbau ihres Service-Portfolios
Kubernetes
Im Xelon HQ könnt ihr innerhalb weniger Minuten Kubernetes Cluster kreieren, auf eure Bedürfnisse anpassen und mit Block Storage sowie Load-Balancing-Funktionen kombinieren. Fokussiert euch auf den Ausbau eurer Services, wir kümmern uns um die Infrastruktur
Windows & Linux
Erstellt komplette Serverinfrastrukturen für eure Kunden und Projekte in unserer Public Cloud oder einer Private Cloud. Xelon HQ vereinfacht nicht nur das Aufsetzen von IT-Infrastrukturen, sondern auch die Verwaltung, Skalierung und Abrechnung

Storage

Im Xelon HQ sind eure Daten sicher: Xelon HQ erfüllt höchste SecurityStandards und bietet regelmässige Backups
Persistent Storage
Dank Persistent Storage werden eure Daten und Systeme in unserer Cloud-Infrastruktur auch bei Strom- oder Internetausfällen gespeichert. So verliert ihr keine wertvollen Arbeitsfortschritte
Block Storage
Im Xelon HQ werden eure Daten in Blöcke aufgeteilt und separat gespeichert (Block Storage). Dies vereinfacht die Verwaltung sowie Konfiguration und erleichtert den Arbeitsalltag von ITDienstleistern
Backup & Snapshots
Erstellt im Xelon HQ jederzeit Backups und Snapshots. Unsere Cloud-Plattform bietet sowohl die tägliche Sicherung kompletter Server als auch ein agentenbasiertes Backup für Files und Applikationen
